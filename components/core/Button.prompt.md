Rectangular CTA button with trailing arrow. Use for all primary and secondary calls-to-action across the site.

```jsx
<Button variant="primary">ENTDECKEN</Button>
<Button variant="secondary" size="sm">MEHR ERFAHREN</Button>
<Button variant="ghost" icon={false}>Alle Weine ansehen</Button>
<Button variant="primary-light">UNSERE WEINE</Button>
<Button variant="secondary-light">MEHR ERFAHREN</Button>
```

Variants: `primary` (black fill), `secondary` (outlined), `ghost` (text-only), `primary-light` (cream fill for dark backgrounds), `secondary-light` (outlined on dark).
Sizes: `sm`, `md` (default), `lg`.
Always uppercase. Trailing arrow can be hidden with `icon={false}`.
